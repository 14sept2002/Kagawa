# IP Tracking Tool
Kagawa is a tool used to extract information from IP addresses version 4 or 6 of people/self/and websites

Powered by `https://ip-api.com/`
# Generated information
- Status
- IP
- Country
- Country code
- Region
- Region name
- City
- ZIP code
- Latitude
- Longitude
- Time zone
- ISP
- Org
- ASN
- Public IP
- Private IP
- Version
- Bit
# Tested on
Termux
# Requirements
Python
# Module installation
`pip install ipaddress`
# Commands
`apt install git`

`git clone https://github.com/14sept2002/Kagawa.git`

`cd Kagawa`

`python kagawa.py`
