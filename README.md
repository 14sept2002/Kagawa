# IP Tracking Tool
Kagawa is a tool used to extract information from IP addresses of people/self/and websites

Powered by https://ip-api.com/
# Generated information
1. Status
2. IP
3. Country
4. Country code
5. Region
6. Region name
7. City
8. ZIP code
9. Latitude
10. Longitude
11. Time zone
12. ISP
13. Org
14. ASN
15. Public IP
16. Private IP
17. Version
18. Bit

Termux
